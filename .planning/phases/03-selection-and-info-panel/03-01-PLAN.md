    NOTE: The `connectedEdgeIds` set above is populated inside the `setEdges` functional update. However, because `setNodes` runs after `setEdges`, the `connectedNodeIds` set needs to be built separately. Use a two-pass approach: first iterate edges to build both sets, then apply to nodes and edges:

    ```typescript
    useEffect(() => {
      if (selectedNodeId === null) {

        setEdges(eds => eds.map(e => ({ ...e, style: { ...e.style, opacity: 1 } })));
        return;
      }
      // Build connected sets from current edges state via ref or by reading nodes/edges directly.
      // Since setNodes/setEdges do not expose current state synchronously, use the nodes/edges
      // variables from useNodesState/useEdgesState (they are in scope in GraphCanvas):
      const connectedNodeIds = new Set<string>([selectedNodeId]);
      const connectedEdgeIds = new Set<string>();
      edges.forEach(e => {
        if (e.source === selectedNodeId || e.target === selectedNodeId) {
          connectedEdgeIds.add(e.id);

        nds.map(n => ({ ...n, style: { ...n.style, opacity: connectedNodeIds.has(n.id) ? 1 : 0.2 } }))

        eds.map(e => ({ ...e, style: { ...e.style, opacity: connectedEdgeIds.has(e.id) ? 1 : 0.2 } }))
    ```
